## Workflow & Tools

### 1. Data cleaning & Preparation
The extracted dataset was processed in Excel for:
- Null cleaning and duplicate removal.
- Preliminary analysis of distributions.

### 2. Data Extraction (MySQL)
- Using SQL queries to select relevant fields.
- Preliminary analysis of distributions.
- Data aggregation to optimize loading in subsequent stages.

### 3. Data Visualization (PowerBI)
- Dax Custom Measures.
- Interactive Matrix: View turnover by department with rules-based conditional formatting for optimal visual contrast.
- Trend Analysis: Line chart for monitoring the evolution of dropouts from 2021 to 2025.

## Key Insights
- Identifying the root cause of abandonment.
- Identify department with the highest turnover rate relative to average performance.
- Correlation between years of seniority and probability of resignation.
